⚙ Settings

Add a Claude API key to enable automatic meter reading from photos. The key is stored only in your browser — never sent anywhere except Anthropic's API.

How it works

After taking a photo, tap ✨ Read Meter with AI. The photo is sent to Claude (a fast AI model) which reads the number off the gauge, dial, or display and fills it in automatically. Works on analog gauges, digital displays, dipsticks, flow meters — most things with a number on them.

Cost: roughly $0.001 per photo read (fractions of a cent).

No key? No problem — just type the number in manually.
